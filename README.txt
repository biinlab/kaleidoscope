Kaleidoscope
============

Un projet animé par Erasme, dans le cadre du museolab 3



Note the line command to package the app for android :

from python-for-android/dist/default, run the command

./build.py --package org.erasme.kaleidoscope --name kaleidoscope --version 2.0 --dir location_of_the_folder --icon-name kaleidoscope --icon location_of_the_folder/icon.png --permission INTERNET debug
